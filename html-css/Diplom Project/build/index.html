Course [Business Analysis in IT]

is a comprehensive programme that allows you to master required techniques within 3 months to work as a Business Analyst in large companies.

[After the course, you can]:

  • To model the main business processes in the implementation of IT projects
  • Prioritize requirements and draw up appropriate types of project documentation
  • Identify project risks and learn the basics of risk management
  • Fill out the value proposition templates
  • Develop prototypes and use the basic UI / UX skills

[Requirements for potential students]:

  • Negotiation skills and ability to formulate ideas and thoughts verbally
  • Technical skills or desire to gain such skills
  • Upper-Intermediate level of English

    Introduction

    • Expectations,

    Introduction into Business Analysis
    • Necessary competencies and best practices of BAs- (Listening, Facilitation Skills, Leadership & Communication Skills)

    Software Development Process
